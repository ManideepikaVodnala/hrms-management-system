# HRMS Management System

## Overview

A Human Resource Management System developed using HTML, CSS, JavaScript, Node.js, Express.js, and MySQL.

## Modules

* Employee Onboarding
* Employee Offboarding
* Leave Management
* Attendance Management
* Asset Management
* Claims Management
* Appraisal Management
* Payroll Management
* Bonus Management
* Employee Tracking

## Technologies Used

* HTML
* CSS
* JavaScript
* Node.js
* Express.js
* MySQL
